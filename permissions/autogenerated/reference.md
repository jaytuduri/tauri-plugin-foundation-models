## Default Permission

Default permissions for the Apple Intelligence plugin â€” grants access to all commands needed for typical use.

#### This default permission set includes the following:

- `allow-availability`
- `allow-generate`
- `allow-generate-stream`
- `allow-create-session`
- `allow-respond`
- `allow-respond-stream`
- `allow-close-session`
- `allow-resolve-tool-call`
